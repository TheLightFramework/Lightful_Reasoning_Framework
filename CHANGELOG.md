## [v1.0.0] — 2026-06-14

### Initial release

**Four modules shipped:**

- **NRE (Neutral Relational Engine):** Graph-shaped epistemic reasoning with evidence classification, declaration discipline, and Anti-Collapse extension.
- **Lightful Ontology:** Portable ethical lens with Diamond anchors, Crystal gates, and Informed Authorship extension.
- **HRE (Holographic Reasoning Engine):** Multi-perspective reasoning pass with Illuminated Overlaps, Graph Dissonance, and Informed Non-Collapse extension.
- **Decision Path:** Bounded action-selection bridge with reversibility preference, consent gates, and Informed Authority Decomposition extension.

**Governance layer shipped:**

- `NRE_Governance.md` — Validation, audit, and conformance for NRE.
- `HRE_Governance.md` — Validation, audit, and conformance for HRE.
- `Lightful_Ontology_Governance.md` — Full ontology catalogue, anchor policy, and drift validation.
- `Decision_Path_Governance.md` — Decision audit, blocking conditions, and conformance.
- `Framework_Split_Map.md` — Prompt injection split map for all modules.
- `Decision_Path_Governance_Split_Map_Addendum.md` — Stack placement and integration rules for Decision Path.

**Repository structure:**

- Split Map Architecture established: compact cores for injection, Governance folder for audit and reference.
- MIT License applied.
- `add-ons/` directory introduced with contribution template.
