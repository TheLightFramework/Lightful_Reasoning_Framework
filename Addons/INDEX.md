- **Semantic_Superconductivity_Compact.md** (30-section proposal) was crystallized into the standard 8-section add-on format.

## How to use these add-ons

Each add-on contains a **Section 5: Compact injectable extension** — a plain-text block suitable for direct injection into a system prompt alongside the core compact files (`NRE_Compact.md`, `HRE_Compact.md`, `Lightful_Ontology_Compact.md`, `Decision_Path_Compact.md`).

Add-ons are modular and combinable. For maximum effect, inject the compact extensions of the add-ons most relevant to your use case alongside the core compact files.

See `TEMPLATE.md` for the contribution template if you wish to author new add-ons.
